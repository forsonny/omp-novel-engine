# Seven Point Structure

Use Hook, First Plot Point, First Pinch, Midpoint, Second Pinch, Second Plot Point, Resolution. Work backward from Resolution when designing finite arcs. For indefinite serials, resolve seasons/books/arcs while keeping the serial promise open.

## Required behavior

- Follow the active Story OS workflow constraints.
- Use Story OS MCP for durable canon and memory.
- Stop at human approval gates.
- Never mark a chapter complete without mandatory gate pass and human approval.
